# 코딩 테스트 준비용 알고리즘 공부 일지 (22.07.01~) :smirk:
- 매일 매일 N일차 -> N+1일차 -> N+2일차 순으로 반복
- N일차 : 구현, DFS, 이분탐색, 그리디
- N+1일차 : BFS, 정렬, 시뮬레이션
- N+2일차 : 트리,백트래킹,DP
- 당일에 일이 생겨 하지 못했을 때, 다음 일차로 넘어가지 말고 계속 이어서 진행.
- 그날 그날 풀었던 것과 못 풀었던 것을 기록하고, 간단히 한줄 평으로 핵심을 짚어보자.
